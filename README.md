## Requirements

Follow this link:

https://tailwindcss.com/docs/installation

Add this in package.json under scripts

```json
"tw:build": "tailwindcss build ./css/app.css -o ./views/style.css"
```
To run this:

```sh
$ npm run tw:build 
```
